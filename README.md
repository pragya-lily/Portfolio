# Pragya Rajput Portfolio

A modern, responsive personal portfolio website built using HTML and CSS. It showcases a professional profile with a hero section, about section, education table, skills cards, project highlights, contact form, and social links. The design also includes a light/dark theme toggle and smooth, card-based layouts for a polished user experience.

## Overview

This portfolio is a single-page website created to present personal information, technical skills, certifications, and contact details in a visually appealing format. It is designed to be simple, elegant, and easy to navigate across different screen sizes.

## Features

- Responsive one-page portfolio layout
- Light and dark theme switch using a CSS checkbox toggle
- Profile header with navigation links
- About section with education details in a table format
- Animated skills cards
- Certifications and project showcase section
- Contact form layout
- Footer with LinkedIn and email links
- Clean UI with smooth hover effects and modern styling

## Tech Stack

- HTML5
- CSS3
- Google Fonts: Poppins
